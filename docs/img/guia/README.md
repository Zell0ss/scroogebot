## Consejos para las capturas

- Usa modo claro o oscuro consistente en todas las capturas
- Recorta solo la respuesta del bot, no toda la pantalla del móvil
- Para `analiza-ibe.png` y `analiza-nvda.png`, intenta capturarlas el mismo día para que la comparación sea válida
- Para `backtest.png`, asegúrate de que se ven ambas cestas en el mismo mensaje o captura ambos mensajes seguidos
